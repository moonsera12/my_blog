## 반응형 디자인 주요 특징

### 데스크톱 (>1024px)
- 멀티 컬럼 레이아웃
- 사이드바 고정
- 풀 네비게이션 메뉴
- 고급 필터링 옵션
- 페이지네이션 컨트롤

### 태블릿 (768px - 1024px)
- 2-3 컬럼 그리드
- 컴팩트한 네비게이션
- 드롭다운 메뉴
- 간소화된 필터링

### 모바일 (<768px)
- 단일 컬럼 레이아웃
- 햄버거 메뉴
- 무한 스크롤 또는 "더 보기" 버튼
- 최소화된 필터링 옵션
- 스와이프 가능한 카드

## 컴포넌트 특징

### 포스트 카드
- 썸네일 이미지
- 제목
- 요약 (2줄)
- 메타 정보 (작성자, 날짜)
- 카테고리 태그
- 좋아요 수

### 검색 바
- 자동완성 기능
- 검색어 제안
- 최근 검색어
- 필터링 옵션

### 네비게이션
- 스티키 헤더
- 드롭다운 메뉴
- 반응형 햄버거 메뉴
- 검색 토글

### 댓글 섹션
- 댓글 입력
- 댓글 스레드
- 답글 기능
- 좋아요/싫어요
